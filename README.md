# FamilyPalette（ファミリーパレット）  

## コンセプト  
**「家族の『今』を共有し、『思い出』を宝物にする。」** 10代から60代までが、迷わず、安全に、楽しく使いつづけられる『多世代共生型』スケジュール＆アルバムアプリ。  
  
---
  
## 1. 数値目標. 
* **1年後の累計ダウンロード数:** 200,000 DL.  
* **有料会員（Premium）転換率:** 5%（10,000人）  
* **月額利用料:** 100円（税込）  
* **目標月商:** 1,000,000円  
  
---
  
## 2. 主要機能  
  
### 📅 リアルタイム・カレンダー（色分け表示）  
* **同期:** 誰が予定を追加しても、瞬時に全員の画面に反映される。  
* **視覚化:** ユーザーごとに「パパは青、ママは赤」といった色分けだけでなく、予定の種類（仕事、遊び、記念日）によってアイコンや背景色を自動変化。  
* **技術:** 将来的には Laravel Reverb や Pusher を使って、画面を更新しなくても予定が反映される仕組みを導入。  
  
### 💬 チャット・コメント機能  
* **機能:** 各予定ごとに専用のチャットスペースを設ける。  
* **活用例:** 「買い物：牛乳」という予定に対し、「卵も買ってきて！」「了解！」とやり取り。  
* **メリット:** 予定に関する会話が流れない（LINEのように他の会話に埋もれない）のが最大の利点。  
  
### 🔔 通知・リマインド（記念日・誕生日特化）  
* **通知:** 予定の〇分前通知、記念日や誕生日の1週間前・当日の特別通知。  
* **演出:** 誕生日当日は、アプリを開いた瞬間に画面いっぱいに「おめでとう！」のアニメーションが出る（10代向け）といった工夫。  
  
### ✅ 共有Todoリスト（買い物・持ち物）  
* **機能:** 完了した項目にチェックを入れると、全員のリストから消える（または線が引かれる）。  
* **活用:** 「旅行の持ち物」リストを家族全員で分担してチェック。  
  
### 🖼️ 画像貼り付け（有料版の核）  
* **機能:** 予定に写真を添付（「今日の夕飯」「旅行の思い出」）。  
* **思い出の1枚:** カレンダーのその日の背景に、ベストショットを1枚設定できる機能。
